# my-ai-skills

A collection of AI coding rules and skills for multiple platforms.

## Usage

### Codex CLI

```bash
# Run in your project root
curl -O https://raw.githubusercontent.com/LillianRiv/my-ai-skills/main/<skill-name>/AGENTS.md
codex
```

### Cursor

```bash
curl -O https://raw.githubusercontent.com/LillianRiv/my-ai-skills/main/<skill-name>/.cursorrules
```

### Claude

Upload the `SKILL.md` from the relevant skill directory to your Claude project.
